# Security policy

## Supported versions

Security fixes are provided for the latest public release. Older versions may be asked to upgrade
before a report is investigated.

## Reporting a vulnerability

Use GitHub's private security advisory flow for this repository instead of opening a public issue.
Describe the affected version, expected and observed behavior, and minimal reproduction steps.

Do not include Codex session files, prompts, credentials, cookies, access tokens, private logs,
user-data directories, or proprietary pet assets. Replace identifiers with synthetic values and
attach only the smallest safe reproduction.

## Distribution note

The v1.0.0 Windows installer is not code-signed. Download it only from this repository's GitHub
Releases page and compare its SHA-256 digest with the attached checksum file before running it.
